## Floating point numbers

-   optional sign
-   optional integer part
-   optional decimal part
-   optional exponent part

<Example regex={/^([+-]?(?=\.\d|\d)(?:\d+)?(?:\.?\d*))(?:[eE]([+-]?\d+))?$/g}>
    <li>987</li>
    <li>-8</li>
    <li>0.1</li>
    <li>2.</li>
    <li>.987</li>
    <li>+4.0</li>
    <li>1.1e+1</li>
    <li>1.e+1</li>
    <li>1e2</li>
    <li>0.2e2</li>
    <li>.987e2</li>
    <li>+4e-1</li>
    <li>-8.e+2</li>
    <li>.</li>
</Example>

The positive lookahead `(?=\.\d|\d)` ensures that `.` does not match.

## HSL colours

### Integers from `0` to `360`

-   `360`
-   `300` to `359` — `3`, `[0-5]`, any digit
-   `0` to `299`
    -   optionally `1` or `2` as the hundreds digit
    -   optionally any tens digit
    -   a units digit

<Example regex={/^0*(?:360|3[0-5]\d|[12]?\d?\d)$/g}>
    <li>360</li>
    <li>349</li>
    <li>235</li>
    <li>152</li>
    <li>68</li>
    <li>9</li>
    <li>361</li>
    <li>404</li>
</Example>

### Percentages

-   `100`, optionally followed by `.000…`
-   one or two digit integer, optionally followed decimal part

<Example regex={/^(?:100(?:\.0+)?|\d?\d(?:\.\d+)?)%$/g}>
    <li>100%</li>
    <li>100.0%</li>
    <li>25%</li>
    <li>52.32%</li>
    <li>9%</li>
    <li>0.5%</li>
    <li>100.5%</li>
    <li>42</li>
</Example>

### Bringing it all together

<Example
    regex={
        /^hsl\(\s*0*(?:360|3[0-5]\d|[12]?\d?\d)\s*(?:,\s*0*(?:100(?:\.0+)?|\d?\d(?:\.\d+)?)%\s*){2}\)$/gi
    }
>
    <li>hsl(0,20%,100%)</li>
    <li>HSL(0350, 002%,4.1%)</li>
    <li>hsl(360,10% , 0.2% )</li>
</Example>

## Javascript comments

`[\s\S]` is a hack to match any character _including newlines_.

<Example regex={/\/\*[\s\S]*?\*\/|\/\/.*/g}>
    <li>const a = 0; // comment</li>
    <li>/* multiline */</li>
</Example>

## 24-Hour Time

<Example regex={/^([01]?[0-9]|2[0-3]):[0-5][0-9](:[0-5][0-9])?$/g}>
    <li>23:59:00</li>
    <li>14:00</li>
    <li>23:00</li>
    <li>29:00</li>
    <li>32:32</li>
</Example>
